# ForenIntel

ForenIntel is a command-line based Digital Forensic Intelligence framework
designed to support case-based intelligence gathering, analysis, and reporting
using publicly available open-source information.

## Project Type
Master's Mini Project

## Domain
Digital Forensic Intelligence

## Key Features
- CLI-based execution for forensic reproducibility
- Case creation and case loading
- Evidence and intelligence isolation per case
- Forensic logging for chain of custody
- Modular and extensible architecture

## System Requirements
- Operating System: Ubuntu 24.04 LTS
- Python Version: Python 3.12
- Interface: Command Line Interface (CLI)

## Installation
1. Clone or copy the project directory
2. Create and activate a Python virtual environment
3. Install required Python packages (if any)
